# C++ — Exercícios e estruturas de dados (PT-BR)

Esta pasta reúne exemplos em C++ usados para praticar sintaxe, programação orientada a objetos e estruturas de dados.

Função do conteúdo:
- Conter implementações de estruturas clássicas (listas, filas, pilhas, árvores binárias) e pequenos exercícios de sintaxe.

Subpastas importantes:
- `Binary Tree/` — implementações e exercícios com árvores binárias (inserção, travessias, altura, etc.).
- `dataStructures/` — listas ligadas, pilhas, filas e implementações utilitárias.
- `programming-language-I/` — exemplos de sintaxe, ponteiros, arrays e exercícios básicos.

Como compilar/executar (exemplo rápido):
```bash
# compilar um arquivo cpp
g++ -std=c++17 arquivo.cpp -o arquivo && ./arquivo
```

Sugestões de estudo:
- Abra os arquivos nas subpastas e execute os exemplos individualmente.
- Converta implementações para usar templates quando fizer sentido (generalizar listas/pilhas).

Se quiser, posso gerar README detalhados por subpasta (árvore, estruturas, sintaxe).
