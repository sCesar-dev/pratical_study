# C — Exercícios e exemplos (PT-BR)

Repositório de exemplos em linguagem C usado para estudo de sintaxe, algoritmos e pequenas estruturas de dados.

Função do conteúdo desta pasta:
- Reunir pequenos exercícios resolvidos (ex.: média, operações básicas, conversões entre bases).
- Servir como material de prática para iniciantes em C.

Como compilar/executar (exemplo rápido):

```bash
gcc exemplo.c -o exemplo && ./exemplo
```

Arquivos importantes (resumo):
- `sintax-exercicies/3-values.c` — média ponderada (entrada/saída básica).
- `sintax-exercicies/3-values-if-matriz.c` — variação com if e matriz.
- `sintax-exercicies/altura-simples.c` — cálculo de altura (entrada/saída).
- `sintax-exercicies/fatorial.c` — cálculo de fatorial (loop/recursão explorar).
- `sintax-exercicies/fibonacci.c` — sequência de Fibonacci.
- `sintax-exercicies/imc-simples.c` — cálculo de IMC (exemplo de float/double).
- `sintax-exercicies/palindrome.c` — verificação de palíndromo em strings.
- `sintax-exercicies/decimal-to-binary.c` — conversão decimal -> binário.
- `sintax-exercicies/binary-to-decimal.c` — conversão binário -> decimal.
- `sintax-exercicies/QuadradoMagico.c` — exercício sobre quadrado mágico.
- `sintax-exercicies/soma-simples.c` — soma de dois números (exemplo básico I/O).

Boas práticas de estudo:
- Compile cada exemplo separadamente e leia os comentários dentro dos arquivos.
- Extraia funções e escreva testes simples quando possível.

Contribuições:
- Adicione novos exercícios em `sintax-exercicies/` com nomes descritivos.
